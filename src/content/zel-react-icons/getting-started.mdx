## installation

```
yarn add @zlab-de/zel-react-icons
```

## How to use these icons

```
import React from 'react'
import ZepIconEssentialStar from '@zlab-de/zel-react-icons/ZepIconEssentialStar'

function MyComponent(){
    return(
        <div>
            <ZepIconEssentialStar />
        </div>
    )
}

export default MyComponent
```

## Available Icons

All icons available can be found under the Guidelines Section (Iconography). However, please note, all icons in the zel-react-icon library are camelcased.

For example the icon 'zep-icon-location-pin' can be accessed in this library as '@zlab-de/zel-react-icons/ZepIconLocationPin'.

## Styling

You may change the color and size with the following css propertires;
fill:
width:

```
import React from 'react'
import ZepIconEssentialStar from '@zlab-de/zel-react-icons/ZepIconEssentialStar'

function MyComponent(){
    return(
        <div>
            <ZepIconEssentialStar styles={{fill: 'green', width: 50}} />
            <ZepIconEssentialStar size={50} styles={{color: 'green'}} />
        </div>
    )
}

export default MyComponent
```

```
yarn add @zlab-de/zel-react
```

## How to use our components

You must bring in **zel.css** and select a theme (theme-zeppelin, theme-cat, or theme-rental) to apply the styles from Zeppelin Element Library. It only needs to be brought it once at the root of your application.

```jsx
// index.js
import React from "react";
import ReactDOM from "react-dom";
import App from "./App";

const rootElement = document.getElementById("root");
ReactDOM.render(<App />, rootElement);

// App.js
import React from "react";
import "@zlab-de/zel-react/zel.css"

export default function App() {
  return (
    <div className="theme-zeppelin">
      <h1>Hello World!</h1>
    </div>
  );
}
```

## guidelines.json

This repository also has a **guidelines.json** file available for you to help with styling. This file contains the following infomation (see the guidelines section for more information); colors, spacing, and typography specifications.

Example with React-JSS Theming

```jsx
// index.js
import React from "react";
import ReactDOM from "react-dom";
import App from "./App";
import {ThemeProvider} from 'react-jss'
import theme from '@zlab-de/zel-react/guidelines.json'

const rootElement = document.getElementById("root");
ReactDOM.render(
  <ThemeProvider theme={theme}>
    <App />
  </ThemeProvider>,
  rootElement
);

// App.js
import React from "react";
import "@zlab-de/zel-react/zel.css"
import MyComponent from "./MyComponent"

export default function App() {
  return (
    <div className="theme-zeppelin">
      <MyComponent />
    </div>
  );
}

// MyComponent.js
import React from "react";
import { makeStyles } from '@material-ui/core/styles'

const useStyles = makeStyles(theme => ({
  p: {
    marginBottom: `${ theme.space.xxl.rem }rem`,
    color: theme.indigo.primary,
  },
})

function MyComponent() {
  return (
      <p className={classes.p}>
        Hello World!
      </p>
  );
}

export default withStyles(styles)(MyComponent)
```

## Props

Any alternative props that you wish to pass to a component (ie inline styles) will always be passed to the outer element of that component. For example, see the code for Tag below.

```js
import React from 'react';
import PropTypes from 'prop-types';
import clsx from 'clsx';
import ZepiconsClose from '@zlab-de/zel-react-icons/ZepiconsClose';

const Tag = ({ className: classNameProp, children, onClose, ...other }) => {
  return (
    <span className={clsx(classNameProp, 'zep-tag')} {...other}>
      {children}
      <button onClose={onClose} aria-label="close" className="zep-tag__button">
        <ZepiconsClose className="zep-tag__icons" />
      </button>
    </span>
  );
};

Tag.propTypes = {
  className: PropTypes.oneOfType([PropTypes.object, PropTypes.string]),
  children: PropTypes.string
};

export default Tag;
```

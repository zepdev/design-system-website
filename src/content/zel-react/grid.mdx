# Grid

The Zeppelin grid system provides a mobile-first and flexible layout system. The content is divided into an 12-column grid with defined breakpoints ensure optimized responsiveness and usability on a range of different devices. This overview helps designers and developers deliver clear, functional layouts.

## Example

<div style={{ marginBottom: 30 }}>
  <GridExample />
</div>

```
import React, {useState} from 'react'
import GridContainer from '@zlab-de/zel-react/GridContainer'
import GridCol from '@zlab-de/zel-react/GridCol'
import GridRow from '@zlab-de/zel-react/GridRow'

function MyComponent(){
    return(
        <div>
            <GridContainer>
                <GridRow align="flex-end">
                    <GridCol xxs={4} xs={4} md={8} lg={12}>Col 1</GridCol>
                </GridRow>
            </GridContainer>
        </div>
    )
}

export default MyComponent
```

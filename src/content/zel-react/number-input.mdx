# Number Input

The Number Input is a basic counter component to be used on forms to increase or decrease an incremental value. Always use a clear and concise label for the number input and allow the user to also choose to type a number in the field.

## Example

<div style={{ marginBottom: 30 }}>
  <NumberInputExample />
</div>

```
import React, {useState} from 'react'
import NumberInput from '@zlab-de/zel-react/NumberInput'

function MyComponent(){
    const [count, setCount] = useState(0)

    return(
        <div>
            <NumberInput
                label="demo number input"
                count={count}
                onChange={value => setCount(value)}
                onMinus={() => setCount(count - 1)}
                onPlus={() => setCount(count + 1)}
            />
        </div>
    )
}

export default MyComponent
```

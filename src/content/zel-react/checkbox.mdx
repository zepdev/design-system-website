# Checkbox

With checkboxes, users can select multiple options in a list of options. Use checkboxes when looking for yes or no answers. Users should be able to select the checkbox by clicking on the box directly or by clicking on its label. Checkboxes must always contain a clear and explicit label which appear on the right side of the checkbox.

## Example

<div style={{ marginBottom: 30 }}>
  <Checkbox label="Checkbox" />
</div>

```
import React, {useState} from 'react'
import Checkbox from '@zlab-de/zel-react/Checkbox'

function MyComponent(){
   const [state, setState] = React.useState({
        isCheckedA: true,
        isCheckedB: true,
        isCheckedF: true,
    });

    const handleChange = name => event => {
        setState({ ...state, [name]: event.target.checked });
    };

    return(
        <form>
            <Checkbox
                label="A"
                isChecked={state.isCheckedA}
                handleChange={handleChange('isCheckedA')}
            />
            <Checkbox
                label="B"
                isChecked={state.isCheckedB}
                handleChange={handleChange('isCheckedB')}
            />
            <Checkbox
                label="C"
                isChecked={state.isCheckedC}
                handleChange={handleChange('isCheckedC')}
            />
        </form>
    )
}

export default MyComponent
```

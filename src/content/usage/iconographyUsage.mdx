# Developers

Icons in SVG format and as an SVG sprite are available with the zeppelin-element-library. However, all icons in svg format may also be downloaded as a for individual use from the iconography page if your project only requires a select number of icons.
All individual icons can be found in the folder **'zeppelin-element-library/assets/icons/SVG/{zepicons-iconname.svg}'** and the SVG sprite in **'zeppelin-element-library/assets/icons/sprite.svg'** (you must also include 'zeppelin-element-library/assets/icons/svgxuse.min.js', see below).

## Using icons as an SVG

Example as a React Component

```
// CloseIcon.jsx
import React from 'react'
import PropTypes from 'prop-types'

const CloseIcon = ({ className, width, height, fill, viewBox, ariaLabel }) => (
  <svg
    width={width}
    height={height}
    xmlns="http://www.w3.org/2000/svg"
    viewBox={viewBox}
    aria-labelledby="title"
    className={className}
  >
    <title id="title_copy">{ariaLabel || 'Close'}</title>
    <polygon fill={fill} points="19 6.41 17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12"/>
  </svg>
)

CloseIcon.propTypes = {
  width: PropTypes.string,
  height: PropTypes.string,
  viewBox: PropTypes.string,
  fill: PropTypes.string,
  ariaLabel: PropTypes.string,
}

CloseIcon.defaultProps = {
  width: '32',
  height: '32',
  viewBox: '0 0 24 24',
  fill: 'currentColor',
}

export default CloseIcon
```

## Using SVG Sprite

To use the SVG Sprite in basic html please download the SVG Sprite file bundle from the Overview Section. The SVG Sprite file must be copied into the body of your html. Make sure that the svgxuse.js file is also added where you add your javascript files.

We reccomend using the SVG Sprite from our CDN however you may download and use it yourself.

CDN:

```html
<!DOCTYPE >
<html lang="en">
  <head>
    ...
  </head>
  <body>
    <svg>
      <use
        xlink:href="https://cdn-zel.zepdev.net/zel/{{ zelVersion }}/css/assets/icons/sprite/symbol-defs.svg#zepicons-account"
      ></use>
    </svg>
  </body>
  <script defer src="svgxuse.js"></script>
</html>
```

Self Hosted:

```html
<!DOCTYPE >
<html lang="en">
  <head>
    ...
  </head>
  <body>
    <!-- SVG Sprite -->
    <svg width="0" height="0" class="hidden">
      <symbol
        xmlns="http://www.w3.org/2000/svg"
        viewBox="0 0 24 24"
        id="zepicons-account"
      >
        <path
          d="M6 17c0-2 4-3.1 6-3.1s6 1.1 6 3.1v1H6v-1zm9-8a3 3 0 1 1-6 0 3 3 0 0 1 6 0zM3 5v14a2 2 0 0 0 2 2h14a2 2 0 0 0 2-2V5a2 2 0 0 0-2-2H5a2 2 0 0 0-2 2z"
        ></path>
      </symbol>
    </svg>

    <svg>
      <use xlink:href="#zepicons-account"></use>
    </svg>
  </body>
  <script defer src="svgxuse.js"></script>
</html>
```

SVG sprites in React: The SVG sprite is available at 'zeppelin-element-library/assets/icons/sprite.svg'. You must also add the file 'zeppelin-element-library/assets/icons/svgxuse.min.js' for Internet Explorer compatibility (it uses already available .png files in place of the svg files as a fallback).

```jsx
//Apps.js
import 'zeppelin-element-library/assets/icons/sprite/svgxuse.min.js' // ie fallback for sprites

// Icon.jsx
import React from 'react'
import PropTypes from 'prop-types'
import svg from 'zeppelin-element-library/assets/icons/sprite/sprite.svg'

const Icon = ({ className, width, height, icon, viewBox, ariaLabel }) => {
  return (
    <svg
      width={width}
      height={height}
      xmlns="http://www.w3.org/2000/svg"
      aria-label={ariaLabel}
      viewBox={viewBox}
      className={className}
      fill="currentColor"
    >
      <use xlinkHref={`${svg}#${icon}`} />
    </svg>
  )
}

Icon.propTypes = {
  width: PropTypes.string,
  height: PropTypes.string,
  viewBox: PropTypes.string,
  icon: PropTypes.string.isRequired,
  ariaLabel: PropTypes.string,
  className: PropTypes.object
}

Icon.defaultProps = {
  width: '32',
  height: '32',
  viewBox: '0 0 24 24',
}

export default Icon

// MyTagComponent.jsx
import React from 'react'
import Icon from '../icon/Icon';

function MyTagComponent() {
  return (
      <span role="button" className="zep-tag">
        Tag
        <Icon icon="zepicons-close" />
      </span>
  );
}

export default MyTagComponent;
```

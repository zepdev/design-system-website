# Download

### Compiled CSS

If you would like to download and use these styles for your own purposes directly in your project you may download the minified file below.

### CDN

Use the cached version of our files from our CDN

```js
<link rel="stylesheet" href="https://cdn-zel.zepdev.net/zel/css/zeppelin-element-library.css" integrity="XXX" crossorigin="anonymous">
<script src="https://cdn-zel.zepdev.net/zel/css/zeppelin-element-library.min.js" integrity="XXX" crossorigin="anonymous"></script>

```

Use the gzipped cached version of our files from our CDN

```js
<link rel="stylesheet" href="https://cdn-zel.zepdev.net/zel/css/zeppelin-element-library.css.gz" integrity="XXX" crossorigin="anonymous">
<script src="https://cdn-zel.zepdev.net/zel/css/zeppelin-element-library.min.js.gz" integrity="XXX" crossorigin="anonymous"></script>

```

### NPM Package

Use Zellepin Elements Libary's source code using your favorite package manager.

```js
yarn add zeppelin-element-library
```

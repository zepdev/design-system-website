## installation

```js
yarn add @zlab-de/zel-vue-icons
```

## How to use these icons

```vue
<template>
  <div>
    <ZepiconsCart />
  </div>
</template>

<script>
import ZelVueIcons from '@zlab-de/zel-vue-icons'
import ZepiconsCart from '@zlab-de/zel-vue-icons/zepicons-cart'

export default {
  name: 'App',
  components: {
    ZepiconsCart,
  },
}
</script>
```

## Available Icons

All icons available can be found under the Guidelines Section (Iconography).

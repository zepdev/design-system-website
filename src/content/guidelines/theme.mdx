# Theme

There is no base color for the zeppelin-element-library, almost all elements require that a theme class is present.

The theme class can be add to any html tag (`<main>`,`<body>`,`<div>`...) as long it wraps all elements you want to use.

```js
<main class="theme-zeppelin" id="app-root">
  ...
</main>
```

We currently offer three main themes;

```js
// Zeppelin Indigo Blue
<div class="theme-zeppelin" >...</div>

// CAT Yellow
<div class="theme-cat" >...</div>

// Rental Red
<div class="theme-rental" >...</div>
```

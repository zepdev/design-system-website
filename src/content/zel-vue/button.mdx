# Button

Buttons are items used to express and initialize an action when the user clicks or touches it. The Zeppelin Button has two main styles, primary and secondary and requires the use of a theme. Each page can have one or two primary buttons.

## Example

```js
<template>
  <div id="my-component">
    <zel-vue-button variant="primary" @click="onHandleClick">Primary Button</zel-vue-button>
  </div>
</template>

<script>
import ZelVueButton from "@zlab-de/zel-vue";

export default {
  name: "my-component",
  components: {
   ZelVueButton,
  },
  data: function() {
    return {
      testmessage: "You Clicked the Button!",
    };
  },
  methods: {
    onHandleClick() {
      alert(this.testmessage);
    },
  }
};
</script>
```

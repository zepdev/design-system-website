# Number Input

The Number Input is a basic counter component to be used on forms to increase or decrease an incremental value. Always use a clear and concise label for the number input and allow the user to also choose to type a number in the field.

## Example

```js
<template>
  <div id="my-component">
    <zel-vue-number-input
      minustitle="Zeppelin Minus"
      plustitle="Zeppelin Plus"
      v-model="counterparent"
      :max="10"
      :min="5"
      :step="2"
      @onminus-click="minusClick"
      @onplus-click="plusClick"
    />
  </div>
</template>

<script>

import { ZelVueNumberInput } from "@zlab-de/zel-vue";

export default {
  name: "my-component",
  components: {
    ZelVueInput
  },
  data: function() {
    return {
      counterparent: 0,
    };
  },
  methods: {
    minusClick(evtValue) {
      this.counterparent = evtValue;
    },
    plusClick(evtValue) {
      this.counterparent = evtValue;
    },
  }
};
</script>
```
